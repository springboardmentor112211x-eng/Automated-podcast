## Project Overview

Podcast transcription and segmentation tool. The project is organized into three main components:

- **src/transcription/** - Convert audio to text
- **src/segmentation/** - Segment transcripts (e.g., by speaker, topic, or time)
- **src/audio_analysis/** - Audio processing and analysis utilities
- **tests/** - Test suite
